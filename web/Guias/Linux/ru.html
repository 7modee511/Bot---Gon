Установка на Linux

Команды

Откройте Терминал или Konsole и вставьте команды в соответствии с вашей дистрибуцией.

Debian и производные

sudo apt install wget curl nodejs nano npm git ffmpeg python3 -y

Arch и производные

sudo pacman -Syu wget curl nodejs nano npm git ffmpeg python --noconfirm

Fedora

sudo dnf install -y wget curl nodejs nano npm git ffmpeg python3

После завершения выполните это

git clone https://github.com/BrunoSobrino/TheMystic-Bot-MD.git mystic && cd mystic && npm install && npm start .

После этого должно начаться, запросив номер для входа в систему

Чтобы перезапустить, выполните в Терминале это

cd mystic && npm start
