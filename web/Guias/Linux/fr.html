Installation sur Linux

Commandes

Ouvrez le Terminal ou Konsole et collez les commandes en fonction de votre distribution.

Debian et dérivés

sudo apt install wget curl nodejs nano npm git ffmpeg python3 -y

Arch et dérivés

sudo pacman -Syu wget curl nodejs nano npm git ffmpeg python --noconfirm

Fedora

sudo dnf install -y wget curl nodejs nano npm git ffmpeg python3

Une fois cela terminé, exécutez ceci

git clone https://github.com/BrunoSobrino/TheMystic-Bot-MD.git mystic && cd mystic && npm install && npm start .

Après cela, il devrait démarrer en demandant le numéro pour se connecter

Pour redémarrer, exécutez ceci dans le Terminal

cd mystic && npm start
