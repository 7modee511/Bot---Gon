Instalação no Linux

Comandos

Abra o Terminal ou Konsole e cole os comandos de acordo com sua distribuição.

Debian e derivados

sudo apt install wget curl nodejs nano npm git ffmpeg python3 -y

Arch e derivados

sudo pacman -Syu wget curl nodejs nano npm git ffmpeg python --noconfirm

Fedora

sudo dnf install -y wget curl nodejs nano npm git ffmpeg python3

Depois que isso terminar, execute o seguinte

git clone https://github.com/BrunoSobrino/TheMystic-Bot-MD.git mystic && cd mystic && npm install && npm start .

Depois disso, deve iniciar solicitando o número para fazer login

Para reiniciar, execute isto no Terminal

cd mystic && npm start
